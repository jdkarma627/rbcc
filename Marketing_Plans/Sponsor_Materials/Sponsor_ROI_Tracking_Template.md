# RBCC Sponsor ROI Tracking Template (v2)

## How to Use This

Create one copy of this file per partner. Fill in the Sponsor Profile when the agreement is signed. Log every delivered activation in the Delivery Log as it happens — do not wait until end of season. Use the Monthly Summary to track pacing, and the Quarterly Review to prepare partner-facing reports.

Target delivery rate for all paid tiers: 90% or above.

## KPI Definitions

| Term | Definition |
| --- | --- |
| Deliverable | A specific visibility action promised in the signed agreement (e.g., one match-day social post, one newsletter mention) |
| Evidence | A verifiable record that the deliverable was completed — a link, screenshot, photo, or document |
| Planned | The number of deliverables committed for the period in the signed agreement |
| Delivered | The number of deliverables completed with evidence recorded |
| Completion % | Delivered / Planned, expressed as a percentage |
| Exception | Any planned deliverable that was not completed as scheduled, with documented reason and make-up plan |
